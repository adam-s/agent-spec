# CSV Reporter

A Python CLI tool that reads sales data from a CSV file and prints summary statistics.

## Usage

```bash
python3 report.py
```

## Output

Prints five statistics:
1. Total Revenue
2. Top 3 Products by Units Sold
3. Revenue by Region (descending)
4. Highest Revenue Month
5. Highest Average Order Value by Product

## Testing

```bash
python3 test.py
```
