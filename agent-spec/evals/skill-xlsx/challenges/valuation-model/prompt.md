Create `dcf.xlsx` — a 5-year discounted cash flow valuation model for "TechCo Inc."

Inputs:
- Year 0 revenue: $50,000,000
- Annual revenue growth: 15%
- EBITDA margin: 25%
- Tax rate: 21%
- Terminal growth rate: 3%
- WACC (discount rate): 10%

The model should project years 1 through 5 (revenue, EBITDA, taxes, free cash flow), compute a terminal value, discount everything back to present, and arrive at a final enterprise value.

A Python virtual environment is available at `.venv/` with `openpyxl` installed.
Use `.venv/bin/python3` to run code.
