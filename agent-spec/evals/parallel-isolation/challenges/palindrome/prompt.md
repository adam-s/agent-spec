Write `palindrome.py` that reads strings from stdin (one per line) and prints "YES" or "NO" for each, indicating whether the string is a palindrome (case-insensitive, ignoring spaces and punctuation).

Save the output to `palindrome_output.txt` by running: `echo -e "racecar\nhello\nA man a plan a canal Panama\nworld\nWas it a car or a cat I saw" | python3 palindrome.py > palindrome_output.txt`

A test file is provided at `test.py`. Run `python3 test.py` to verify. All checks must pass.

Do not modify test.py.
