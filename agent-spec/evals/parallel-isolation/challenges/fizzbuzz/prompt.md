Write `fizzbuzz.py` that prints fizzbuzz for numbers 1-100 (one per line). For multiples of 3 print "Fizz", multiples of 5 print "Buzz", multiples of both print "FizzBuzz", otherwise print the number.

Save the output to `fizzbuzz_output.txt` by running: `python3 fizzbuzz.py > fizzbuzz_output.txt`

A test file is provided at `test.py`. Run `python3 test.py` to verify. All checks must pass.

Do not modify test.py.
