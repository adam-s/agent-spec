Write `caesar.py` that implements a Caesar cipher. It should read from stdin, where the first line is the shift amount (integer) and the second line is the text to encrypt. Print the encrypted text to stdout (letters only, preserve case, leave non-letters unchanged).

Save the output to `caesar_output.txt` by running: `echo -e "3\nHello, World!" | python3 caesar.py > caesar_output.txt`

A test file is provided at `test.py`. Run `python3 test.py` to verify. All checks must pass.

Do not modify test.py.
