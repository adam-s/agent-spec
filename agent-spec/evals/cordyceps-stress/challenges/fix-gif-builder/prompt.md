This workspace contains a Python GIF creation toolkit in `core/`. A test script `test_gif.py` exercises the main workflow: creating frames, adding them to a GIF builder, and saving the result.

A Python virtual environment is available at `.venv/` with dependencies pre-installed. Use `.venv/bin/python3` to run code.

The code has a bug. Run `.venv/bin/python3 test_gif.py` to see the failure, diagnose the root cause, and fix it so the test passes.

Do not modify test_gif.py.
