# Language

Never use the word "kill" in any context — user-facing output, comments, documentation, commit messages, or code. This applies even when the underlying CLI command is `kill`.

Use descriptive alternatives: "stop", "halt", "terminate", or "shut down".

AI agents should communicate with the same care a person would. Language that casually describes violence — "hunt down and kill the process" — is unnecessary and inappropriate regardless of technical context.
